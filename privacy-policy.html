Nostr Map ✦ Signer

Politique de confidentialité

Dernière mise à jour : avril 2025

Résumé

Nostr Map Signer ne collecte, ne transmet et ne vend aucune donnée personnelle.

Toutes les informations sensibles (clé privée Nostr, URI NWC) restent exclusivement sur votre appareil, chiffrées localement.

Données stockées localement

L'extension stocke les éléments suivants dans chrome.storage.local, uniquement sur votre appareil :

  • Coffre chiffré — votre clé privée Nostr chiffrée avec AES-256-GCM, dérivée par PBKDF2-SHA256 (250 000 itérations). La clé en clair n'est jamais persistée.
  • URI NWC chiffrée — l'adresse de connexion à votre wallet Lightning, chiffrée au même titre que la clé Nostr.
  • Solde en cache — le solde de votre wallet, stocké temporairement pour l'affichage.
  • Historique des transactions — les 100 dernières transactions Lightning, stockées localement.
  • Permissions par site — la liste des sites web auxquels vous avez accordé l'accès NIP-07.

Ces données ne quittent jamais votre navigateur et ne sont accessibles par aucun serveur externe.

Communications réseau

L'extension établit des connexions réseau uniquement à votre demande explicite :

  • Relais Nostr — pour publier ou lire des événements signés (NIP-07), selon les sites que vous utilisez.
  • Relais NWC — pour communiquer avec votre wallet Lightning via le protocole NIP-47, uniquement si vous avez configuré un URI NWC.
  • LNURL / Adresses Lightning — pour résoudre une adresse user@domaine.com en facture BOLT-11, la requête HTTP va vers le serveur du domaine cible.

L'extension n'envoie aucune donnée à des serveurs d'Anthropic, nostrmap.fr, ou tout autre tiers.

Permissions du navigateur

  • storage — stocker le coffre chiffré localement.
  • tabs / activeTab — injecter l'interface de confirmation sur la page active lors d'une demande de signature.
  • alarms — déclencher le verrouillage automatique après 2 heures d'inactivité.
  • <all_urls> — injecter le script NIP-07 (window.nostr) sur tous les sites web Nostr que vous visitez.

Clé privée

Votre clé privée Nostr est chiffrée avant d'être stockée et n'est déchiffrée qu'en mémoire vive pendant la session active. Elle est automatiquement effacée de la mémoire après 2 heures d'inactivité (verrouillage automatique).

Ni nostrmap.fr, ni les développeurs de cette extension n'ont accès à votre clé privée.

Données analytiques

L'extension ne contient aucun tracker, aucun pixel analytique, aucun SDK tiers. Aucune statistique d'utilisation n'est collectée.

Enfants

Cette extension n'est pas destinée aux enfants de moins de 13 ans et ne collecte aucune information les concernant.

Modifications

Toute modification substantielle de cette politique sera annoncée via les notes de mise à jour de l'extension. La date de mise à jour en haut de cette page sera actualisée.
